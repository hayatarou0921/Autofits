# Autofits

![Badge Status](https://ci-as-a-service)

自分の服の中からコーディネートを作成し、服装を自動で決められるアプリです。
朝のお出かけ前の服装に悩む時間を減らすことができます。

***DEMO:***

![Demo](https://raw.github.com/wiki/Hayatarou0921/Autfits/images/demoAutfits.gif)

### 主な使用技術
- DB
  -PostgresQL
- 画像アップロード機能
  - CarrierWave
  - MiniMagick
- ユーザー管理機能
  -devise
- ページネーション機能
  - kaminari
- 画像のストレージ管理
  - AWS S3
  - fog-aws
- フロント開発
  -slim
  -bootstrap

- Ruby 2.6.3
- Rails 5.2.3
